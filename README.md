# ZEDLY

Современная образовательная платформа для школ с тестированием учеников.

## Возможности

### Для учеников
- Прохождение тестов по предметам
- Просмотр статистики с графиками (Chart.js)
- Тест на определение интересов
- История прохождения тестов
- Двуязычный интерфейс (Русский/Узбекский)

### Для учителей
- Создание предметов и модулей
- Создание и редактирование тестов
- Просмотр статистики учеников
- Управление классами

### Для администраторов
- Полная панель управления
- Просмотр всех учеников с детальной статистикой
- Управление учителями
- Общая статистика системы
- Автоматические рекомендации

## Технологии

**Frontend:**
- Vite
- Vanilla JavaScript
- Chart.js
- CSS с адаптивным дизайном

**Backend:**
- Express.js
- JWT авторизация
- In-memory хранилище (без БД)
- bcryptjs

## Установка

```bash
# Установка зависимостей
npm run install-all

# Запуск в режиме разработки
npm run dev
```
