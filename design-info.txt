# Информация о дизайне и wireframe приложения ZEDLY

## Общее описание дизайна
Приложение ZEDLY - это веб-приложение для образовательного тестирования с поддержкой ролей: ученик, учитель, администратор. Дизайн адаптивный, с мобильной навигацией внизу экрана. Поддерживает темную и светлую темы.

### Цветовая схема
- **Темная тема (по умолчанию)**:
  - Основной фон: #0A0E1A
  - Вторичный фон: #151B2D
  - Третичный фон: #1F2937
  - Основной цвет: #1b3b6f (синий)
  - Акцент: #2b59c3
  - Успех: #10B981 (зеленый)
  - Опасность: #EF4444 (красный)
  - Предупреждение: #F59E0B (оранжевый)
  - Текст: #F9FAFB (белый), #D1D5DB (серый), #9CA3AF (тусклый)

- **Светлая тема**:
  - Основной фон: #F5F7FA
  - Вторичный фон: #FFFFFF
  - Третичный фон: #F0F3F8
  - Основной цвет: #1b3b6f
  - Акцент: #2b59c3
  - Текст: #0F172A (темный), #334155, #64748B

### Общие элементы интерфейса
1. **Переключатель языка**: Фиксированный в правом верхнем углу, кнопка "RU | UZ"
2. **Нижняя навигация (мобильная)**:
   - Главная (иконка дома)
   - Профиль (иконка пользователя)
   - История (иконка истории)
   - Язык (иконка глобуса)
   - Выход (иконка выхода)
3. **Основной контейнер**: #app, где рендерится контент страниц

## Wireframe основных экранов

### 1. Страница входа (Login Page)
Порядок элементов сверху вниз:
1. Заголовок "ZEDLY"
2. Форма входа:
   - Поле логина
   - Поле пароля
   - Кнопка "Войти"
3. Ссылки: "Забыли пароль?", "Регистрация" (если применимо)

### 2. Выбор роли (Role Selection)
Порядок элементов:
1. Заголовок "Выберите роль"
2. Кнопки выбора роли:
   - Войти как Ученик
   - Войти как Учитель
   - Войти как Администратор

### 3. Смена пароля (Change Password)
1. Заголовок "Сменить пароль"
2. Форма:
   - Текущий пароль
   - Новый пароль
   - Подтверждение пароля
3. Кнопки: "Сохранить", "Отмена"

### 4. Панель ученика (Student Dashboard)
1. Приветствие "Добро пожаловать, [Имя]"
2. Карточки предметов (список доступных предметов)
3. Кнопка "Тест на определение интересов"
4. Статистика: пройденные тесты, средний балл

### 5. Тесты по предметам (Subject Tests)
1. Заголовок предмета
2. Список модулей
3. Для каждого модуля: название, описание, кнопка "Начать тест"

### 6. Модули предмета (Student Subject Modules)
1. Заголовок модуля
2. Описание модуля
3. Список тестов в модуле
4. Кнопки: "Начать тест", "Посмотреть результаты"

### 7. Прохождение теста (Test Taker)
1. Заголовок теста
2. Таймер (если есть)
3. Прогресс-бар
4. Текущий вопрос:
   - Текст вопроса
   - Варианты ответов (радио-кнопки или чекбоксы)
5. Кнопки навигации: "Предыдущий", "Следующий", "Завершить"

### 8. Результаты теста (Test Results)
1. Заголовок "Результаты теста"
2. Балл и процент
3. Детали по вопросам
4. Кнопка "Вернуться к тестам"

### 9. История тестов (Test History)
1. Заголовок "История прохождения тестов"
2. Список пройденных тестов с датами, баллами
3. Кнопки: "Посмотреть детали", "Повторить тест"

### 10. Панель администратора (Admin Dashboard)
1. Статистика: общее количество пользователей, тестов и т.д.
2. Карточки для управления:
   - Предметы
   - Ученики
   - Учителя
   - Тесты

### 11. Управление предметами (Admin Subjects)
1. Список предметов
2. Кнопки: "Добавить предмет", "Редактировать", "Удалить"

### 12. Список учеников (Admin Student List)
1. Фильтры: по классу, школе
2. Таблица учеников: имя, класс, школа, статус
3. Кнопки: "Просмотреть детали", "Редактировать"

### 13. Список учителей (Admin Teacher List)
Аналогично списку учеников, но для учителей.

### 14. Детали ученика (Admin Student Detail)
1. Информация об ученике: имя, класс, школа
2. Результаты тестов
3. Кнопки: "Редактировать", "Удалить"

### 15. Детали учителя (Admin Teacher Detail)
Аналогично деталям ученика.

## Дополнительные элементы
- Модальные окна для подтверждений
- Уведомления (toast messages)
- Загрузчики (spinners) для асинхронных операций
- Графики (Chart.js) для статистики
- Адаптивный дизайн: на мобильных устройствах нижняя навигация, на десктопе возможно боковая панель (но в коде не видно)

## Иконки
Используются Font Awesome иконки:
- fa-home: дом
- fa-user: пользователь
- fa-history: история
- fa-globe: глобус
- fa-sign-out-alt: выход
- И другие для кнопок и элементов

## Адаптивность
- Мобильная навигация внизу
- Фиксированный переключатель языка вверху справа
- Контент адаптируется под размер экрана
